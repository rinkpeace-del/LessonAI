# LessonAI

教材作成AIツールのプロトタイプです。教科・学年・単元・難易度・授業時間をもとに、OpenAI APIで授業プリントや練習問題のたたき台を生成できます。

## 使い方

1. `.env.example` を参考に `.env` を作る
2. `.env` に `OPENAI_API_KEY` を設定する
3. 初回だけ `npm.cmd install` を実行する
4. `npm.cmd start` または `node server.js` を実行する
5. ブラウザで `http://localhost:3000` を開く
6. 教科、学年、単元、条件を入力する
7. 必要なら参考PDFを追加する
8. 「AIで教材を生成」を押す
9. 必要に応じて `Copy` でコピー、PDFボタンで保存する

## .env の例

```env
OPENAI_API_KEY=sk-your-api-key
OPENAI_MODEL=gpt-5.5
PORT=3000
```

## 今できること

- OpenAI APIによる教材本文の生成
- 参考PDFのテキスト抽出と、その内容をもとにした教材生成（テキストPDF向け）
- APIキー未設定時のローカルたたき台表示
- 授業案、ワークシート、小テスト、評価ルーブリックのテンプレート生成
- 生徒用/先生用の出力切り替え
- 生徒用PDF/先生用PDFの出力
- 最近作った教材5件のブラウザ内履歴
- 入力内容に応じた教材タイトルと内容の更新
